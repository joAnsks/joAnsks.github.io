## Game: Pastel Bounce

A retro **brick-breaker / breakout** style mini game.

### Gameplay
- 10 √ó 5 grid of pastel bricks, each showing its point value
- A ball bounces around and must be kept in play using a paddle
- Paddle is controlled via **mouse**, **touch**, or **‚Üê ‚Üí arrow keys**
- **Space** pauses and resumes the game
- Breaking all bricks advances to the next level
- Ball speed increases each level
- Player starts with 3 lives; losing all triggers Game Over
- High score is persisted in `localStorage` under the key `pb_best`

### Scoring
- Bottom row bricks = 10 pts, top row = 50 pts (multiplied by current level)

### Typography
- **Font:** `Press Start 2P` (Google Fonts) ‚Äî pixel/retro style, used everywhere
- Fluid font sizes via `clamp()` to scale across screen sizes

### Layout
- Vertical flex column: header ‚Üí HUD ‚Üí canvas ‚Üí controls hint
- Canvas is responsive: width capped at 560px, height at 60% of viewport
- Overlay (start / pause / game over / level up) sits absolutely over the canvas with a blurred dark backdrop

### Visual Effects
- Animated rainbow gradient on the header (`background-size: 300%` + `@keyframes shimmer`)
- Floating pastel bubbles in the page background (CSS `@keyframes rise`, JS-spawned `<div>` elements)
- Ball motion trail (last 8 positions drawn with fading opacity)
- Ball glow via `createRadialGradient`
- Brick shimmer highlight (sine wave animation per brick)
- Particle burst on brick destruction (14 particles per brick)
